# loopCount

[common]\
var [loopCount](loop-count.md): [Int](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin-stdlib/kotlin/-int/index.html)

Current loop count setting.

- 
   1 = play once (default)
- 
   2 = play twice (repeat once)
- 
   -1 = infinite loop

Can be changed at any time, even during playback.
