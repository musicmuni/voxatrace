//voxatrace/com.musicmuni.voxatrace.sonix.recorder/AndroidAudioRecorder/latestBuffer

latestBuffer

[android]
open override val latestBuffer: AudioBuffer?

The most recently captured audio buffer, if available.

This provides polling-style access for legacy compatibility with patterns like audioFeaturesForNextFrame. For new code, prefer audioStream.

Returns null if no buffer has been captured yet or recording is not active.

Thread-safe for reading from any thread.
